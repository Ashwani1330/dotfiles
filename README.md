## Dotfiles

> Window Manager : **i3**
> <br>
> `i3`, `i3status`, `i3lock`

Main components
---
Session Manager: `lxsession`
<br>
Screenshot tool: `maim`
<br>
Background: `feh`
<br>
Brightness control: `brightnessctl`
<br>
Compositor: `picom`
<br>
Terminal Emulator: `kitty`
<br>
Sound Server: `pulseaudio`
<br>
Font: `Fira Code Nerd Font`
<br>
Application launcher: `rofi`
<br>
Clipboard tool: `greenclip`
